# 交通部中央氣象局每日雨量

### 資料蒐集
 -  資料來源：[交通部中央氣象局每日雨量](http://www.cwb.gov.tw/V7/climate/dailyPrecipitation/dP.htm)
 -  蒐集範圍：2013 ~ 2016

-------------

### 運行方式
 -   安裝 python 2.7
 -   安裝 pip
 -   `pip install -r requirements.txt`
 -   `python Crawler.py 2016`
 -   批次搜集
    - 參考 2013-2016.sh 建立 sh or bat 檔
    - `sh 2013-2016.sh` ( for Mac )
