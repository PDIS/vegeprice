# 蔬菜市場交易價格趨勢

以高雄市公有零售市場交易價格為例，本專案預計呈現各類食品交易價格變化，未來預計結合農委會其他相關資訊（天氣、天災、進口數量等）觀察其變化關係，其中並嘗試以視覺化呈現模式，使批發、銷售以及可能影響事件可比對觀察
